What's New?

  • UPDATE Rewrite of the filtering and selection functions to make the user experience feel faster
  • NEW FEATURE Set a custom download / info URL per-mod
  • NEW FEATURE Support for light and dark mode (dark by default)
  • NEW FEATURE Support for other Farming Simulator Versions (FS13, FS15, FS17, & FS19)
  • NOTICE Consider clearing your mod cache if upgrading from a previous version. It will take a few minutes, but should improve performance. There is a "Clear" button in user preferences.
